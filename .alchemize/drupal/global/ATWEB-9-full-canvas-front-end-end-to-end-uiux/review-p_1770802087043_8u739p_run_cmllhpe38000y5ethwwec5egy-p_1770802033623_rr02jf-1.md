**Issues Found:**
- **Layout tab visible on ineligible nodes**: The "Layout" tab renders on ALL node view pages (including Basic Page which has no exposed slots). Clicking it correctly returns 403 "Access denied". This is a known Drupal core limitation — local task tabs defined with `base_route: entity.node.canonical` render for all entities of that type, regardless of access check results. The route access check properly blocks unauthorized access. This behavior matches the code on `main` and is consistent with how Drupal core's Layout Builder tab also behaves. **Non-blocking.**
